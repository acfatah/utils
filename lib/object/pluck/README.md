# `pluck`

Inspired by Ruby on Rails `pluck` method.

## Usage

```
pluck(arrayOfObject, ...keys, options?)
```

### arrayOfObject

Type: `Array`

An array of objects.

### keys

Type: `string`

The key(s) to extract.

### options?

Type: `object`

#### option.strict

Option to throw an error if provided key is not present.
